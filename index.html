Ben Kimim ?

Yaklaşık 5 yıldır kod yazıyorum 2 yıldır ise web site yazıyorum. İlk olarak oyunlara script yazarak başladım. Verileri nasıl tutarım ve alırım diye düşünürken SQL ile tanıştım. Windows sistemlere uygulama geliştirmeyi öğrenip geliştirdim ama Web site alanın da ilerlemeyi düşünüyorum. İlk öğrendiğim JS library React ondan sonra Next.Js, Node.js. JS libraryi dünyasını daha hızlı bir şekilde keşfetmek için şu anda yapmış olduğumuz site olan. Kodsuzm'un blog sistemini geliştirmeye başladık yakın zamanda tasarımı bittikten sonra Back-End kısmını yazıp bitireceğiz. İşin daha çok Back-End kısmın da olmayı severim. Front-End yazarım fakat pek sevmem kendilerini, tasarım dan anlamam somut bir tasarım var ise Front-End'ni yazarım.
